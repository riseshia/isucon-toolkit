```bash
bin/pull-logs
bin/truncate-logs
```

## Analyze

### access log

use alp.

```
cat access.log | alp json -m '/api/condition/[a-f0-9-]+,/api/isu/[a-f0-9-]+/icon,/api/isu/[a-f0-9-]+/graph,/api/isu/[a-f0-9-]+,/isu/[a-f0-9-]+/graph,/isu/[a-f0-9-]+/condition,/isu/[a-f0-9-]+'
```

### mysql slow log

```
mysqldumpslow slow.log > analyzed.log
```
